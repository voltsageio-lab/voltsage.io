# VoltSage Website (Solid State theme)

Customized VoltSage homepage based on HTML5 UP "Solid State".

## Deploy (GitHub Pages)
1. Create a GitHub repo named `voltsage.io`.
2. Upload all files (the `index.html`, `assets/`, and root files like `CNAME`).
3. In GitHub → Settings → Pages:
   - Source: Deploy from branch
   - Branch: `main`, folder: `/ (root)`
4. In Namecheap → Domain List → Manage → Advanced DNS, add A records:
   - @ → 185.199.108.153
   - @ → 185.199.109.153
   - @ → 185.199.110.153
   - @ → 185.199.111.153
   And CNAME:
   - www → voltsage.github.io
5. Wait for DNS to propagate (10–30 minutes). Visit https://voltsage.io
